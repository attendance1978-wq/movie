# Movie Streaming System

A full-featured movie streaming platform built with Node.js, Express, and MySQL.

## Features

### User Features
- User registration and authentication
- Browse movies by genre, year, and rating
- Search movies
- Watch movies with progress tracking
- Add movies to favorites
- Write reviews and rate movies
- Continue watching from last position
- View watch history

### Admin Features
- Upload new movies
- Manage existing movies
- View system statistics
- User management (planned)

### Technical Features
- Adaptive video streaming
- Secure authentication with JWT
- Responsive design
- Progress saving
- Rating system
- Search and filtering
- Pagination

## Prerequisites

- Node.js (v14 or higher)
- MySQL (v5.7 or higher)
- FFmpeg (for video processing)

## Installation

1. Clone the repository:
```bash
git clone <repository-url>

cd movie-streaming-system

```
